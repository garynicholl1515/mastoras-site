# Mástoras Landing Page

This is the static landing page for Mástoras — Clarity & Story Consultancy.

## Deploying to Netlify
Since this site is built with purely static files (`index.html`, `styles.css`, `script.js`) and no build step is required, deployment is extremely straightforward:

1. **Log into Netlify** (or create an account at netlify.com).
2. Go to the "Sites" tab and click **"Add new site" -> "Deploy manually"**.
3. **Drag and drop** this entire `mastoras-site` folder into the upload box.
4. The site will be live immediately. You can then assign your custom domain in the Site Settings.
